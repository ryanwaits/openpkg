Minimalist theme w/ splash of color. Based on Amp terminal aesthetic.

## Base

- Background: `#1a1a1a`
- Text: `#c5c5c5`

## Tokens

| Token | Hex | Note |
|-------|-----|------|
| Text | `#c5c5c5` | soft gray |
| Variables | `#c5c5c5` | same as text |
| Comments | `#5a5a5a` | dimmed |
| Keywords | `#7ab3a0` | teal accent |
| Punctuations | `#777777` | subtle |
| Strings | `#b8a555` | olive accent |
| Numbers | `#7ab3a0` | teal |
| RegExp | `#b8a555` | olive |
| Functions | `#c5c5c5` | default |
| Operators | `#888888` | mid gray |

## Palette

Only 2 accent colors:
- **Teal** `#7ab3a0` - keywords, numbers
- **Olive** `#b8a555` - strings, regex

Everything else gray tones.
