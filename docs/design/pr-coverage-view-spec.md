## Tab Views

### Changed Tab (Default)
Shows all exports added or modified in this PR.

**Columns**:
- Change type: `+` new, `~` modified, `-` removed
- Export name (linked to source)
- Kind badge (fn, cls, int, type, etc.)
- Coverage bar with percentage
- Status indicator with missing signals

**Sorting**: By file, then by coverage (lowest first)

### Undocumented Tab
Filtered to only new exports without full documentation.
